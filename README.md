# 🕒 Attendance Outing Monitoring Automation 🕒

## 📌 Overview
**Attendance Outing Monitor** is a Python automation project designed to simplify attendance and outing tracking.  
With a clean **PyQt-based GUI**, it automates the process of recording entries and exits, ensuring accurate monitoring with minimal manual effort.  

This tool is ideal for offices, institutions, or events where efficient monitoring of attendance and outings is essential.

---

## ✨ Features
- 🔹 Automated attendance and outing tracking  
- 🔹 User-friendly interface powered by **PyQt**  
- 🔹 Real-time monitoring of in/out records  
- 🔹 Lightweight and easy to deploy  

---

## 🛠️ Tech Stack
- **Python 3.x**  
- **PyQt5** (for GUI)  
- **Automation scripts** for record handling  

---

## 🚀 Installation

1. Clone the repository:
   ```
   git clone https://github.com/sanchayan7432/Campus-Attendance-and-Outing-Monitor-Python-Automation-Project.git
   ```

2. Navigate to the project folder:
 ```
cd attendance-outing-monitoring_automation
```

3. Install dependencies:
```
pip install -r requirements.txt
```
---

## ▶️ Usage- 
1. Run the main script:
```
python app.py
```
2. Use the GUI to:
   
```
- Record attendance
- Track outings
- View logs in real-time
```
